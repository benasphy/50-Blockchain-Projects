# NFT Fractionalization Protocol

## Features

- NFT vault
- Fractional ownership
- ERC20 share tokens
- NFT redemption
- Shared ownership model

## Architecture

NFT Owner
    ↓
Deposit NFT
    ↓
Vault locks NFT
    ↓
Mint ERC20 fractions
    ↓
Fractions become tradable

## Key Concepts

- NFT custody vault
- Fractional ownership
- ERC20 representation
- NFT redemption mechanics

## Flow

1. User deposits NFT
2. Vault locks NFT
3. Protocol mints fractional tokens
4. Fractions can be traded
5. Full owner can redeem NFT
